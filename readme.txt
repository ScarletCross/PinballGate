【操作方法】
・タイトル画面でスペースキーを押すことで、ゲームスタート。
・ゲームオーバー時、スペースキーを押すことで、リトライ。
・左右の矢印キーでそれぞれのフリッパーを操作。
・エンターキーを押すことで、タイトル画面に遷移。


【ソースコード】
PinballGate / Scriptsフォルダ内に当ゲームのソースコードを記載しております。
＊　一部ソースコードは、ビルド後にバージョン管理ツールを使用する際、誤操作により欠落してしまいました。
（Windows版のビルド後、WebGL版をビルドする際にシーンの欠如とともに、ソースコードも紛失しました）
